**Questrag** is an innovative AI-powered SaaS application designed to help users interact with their PDF documents. By combining cutting-edge technologies like **LangChain.js**, **vector embeddings**, and **TypeScript**, Questrag offers precise and conversational responses regarding the content of the uploaded PDFs.

## Features

- **AI-Powered Chat**: Enables users to ask questions about their PDF documents and receive accurate, conversational answers.  
- **PDF Upload and Parsing**: Easily upload PDF documents for interaction.  
- **Vector Embedding Integration**: Uses vector embeddings for semantic search and context-aware responses.  
- **Authentication**: Secure user authentication powered by **Kinde**.  
- **Real-Time Responses**: Quickly processes and responds to queries for an enhanced user experience.  
- **Scalable SaaS Architecture**: Designed to support multiple users and documents efficiently.  

## Technologies  

- **Next.js**: A React framework for building robust, scalable web applications.  
- **LangChain.js**: For creating AI pipelines and conversational interfaces.  
- **tRPC**: Provides type-safe API calls between the server and client.  
- **TypeScript**: Ensures strong typing and better developer experience.  
- **Kinde**: Secure and user-friendly authentication solution.  
- **Vector Embeddings**: For efficient information retrieval and contextual analysis.  

## Getting Started  

To run the project locally, follow these steps:  
```bash
git clone
cd questrag
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.

Questrag represents the future of document interaction by integrating AI-powered conversational capabilities with robust web technologies. This application simplifies accessing and understanding PDF content, making it an essential tool for students, researchers, and professionals.
