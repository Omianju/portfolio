**Dinoysus** is an innovative AI-powered application designed to help users streamline their workflows by analyzing GitHub repositories and summarizing audio meetings. By leveraging cutting-edge technologies, it provides actionable insights and simplifies complex tasks.

## Features

- **GitHub Repository Analysis**: Understand and explore GitHub repositories with AI-driven insights.
- **Audio Meeting Summaries**: Upload meeting recordings to get concise summaries with timestamps.
- **AI-Powered Chat**: Interact with AI to ask detailed questions about uploaded repositories or meeting notes.
- **File Upload Support**: Simple and secure file uploads for audio recordings and repository interactions.
- **Scalable Architecture**: Designed to handle multiple users and large files efficiently.

## Technologies

- **Next.js**: A React framework for building modern, scalable web applications.
- **LangChain.js**: For conversational AI workflows and intelligent query handling.
- **tRPC**: Provides type-safe API calls between the server and client.
- **TypeScript**: Ensures a strong typing system for better maintainability.
- **Octokit**: For interacting with GitHub APIs to extract repository data.
- **Google's Generative AI**: Generates summaries and insights efficiently.
- **pgvector** (via Prisma): Stores vector embeddings for semantic search.
- **Assembly AI**: Transcribes and summarizes audio files.
- **Uploadthing**: Handles secure file uploads for PDFs and audio.

## Getting Started

To run Dinoysus locally, follow these steps:

```bash
git clone
cd dinoysus
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.
